Create my first game using Pygame, with the main goal of becoming proficient in polymorphism and inheritance. The project helped me practice designing classes where multiple game objects share common behavior while still having their own unique actions.

The game is a space battle where the player controls a spaceship that shoots bullets to destroy asteroids moving around the screen. If an asteroid touches the spaceship, the ship explodes and the game ends.

A key mechanic is how asteroid size is handled. When you shoot an asteroid, its behavior changes depending on its size. Large asteroids split into smaller ones, and the smallest asteroids are completely destroyed. This design allowed me to apply inheritance to share common properties between game objects and polymorphism so different objects can respond to actions (like collisions or updates) in their own way.

Overall, the project strengthened my understanding of object-oriented programming, especially how polymorphism and inheritance can simplify game logic and make the code more scalable.
